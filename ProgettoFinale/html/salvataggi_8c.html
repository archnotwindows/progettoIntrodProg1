Descrizione dettagliata

Implementazione del sistema di gestione file (I/O) e persistenza dati.

  • Questo modulo gestisce la creazione, lettura, modifica ed eliminazione dei file di salvataggio. Include anche la logica per la compatibilità tra Windows e Unix e un sistema di riconoscimento pattern per l'attivazione della "Cheat Mode".
